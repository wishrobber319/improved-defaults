# Improved Defaults

Presets the choices on RimWorld's new-game setup screens so you don't have to click through the same options every time. Everything stays fully changeable on the screens — this only sets the starting selection, and each piece degrades gracefully if the relevant mod/DLC isn't loaded.

- **Choose Scenario** — Medieval Overhaul's "Battle Brothers" at the top of the list and pre-selected (instead of Crashlanded).
- **Choose AI Storyteller** — The Antagonist (falls back to Cassandra if that storyteller mod isn't loaded), difficulty "Blood and dust", Reload anytime mode, Anomaly incidents disabled.
- **Create World** — population 5/7, landmarks 6/7, pollution 0%, map size 325×325, plus a custom faction line-up.
- **Colonist timetable** — Recreation at hours 6 and 21.
- **Hostility response** — Attack.
- **Carry (inventory stock)** — Healroot ×3.
- **Reading policy** — Tomes disabled in the default "Anything" policy.
- **Food policy** — trimmed default "Lavish" policy (no rotten/human/insect meat, several categories off with targeted exceptions).
- **Apparel policy** — default "Anything" excludes tainted apparel and King's crown; hit-points 51–100%.
- **Manual priorities** — ON, with a full set of default per-colonist work priorities (flat values plus passion-scaled ones).

Harmony patches on the new-game setup screens and per-pawn initialization; existing saves are unaffected.
