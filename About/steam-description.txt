[b]Stop re-clicking the same new-game setup every time.[/b] Improved Defaults presets the entire new-game flow to how you like it, the instant each screen opens: scenario, storyteller, difficulty, world generation, colonist policies and work priorities. Everything stays fully editable; it only changes the starting selection.

[b]Safe and graceful[/b]
Presets only apply to new games and newly created colonists, so existing saves are never touched, and every piece no-ops if its mod or DLC is missing. Lightweight Harmony patches on the setup screens, no new items and no def bloat, safe to add mid-playthrough.
